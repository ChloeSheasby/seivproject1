Career Types and Salaries in CS

Computer Science is a wildly diverse field, as computers are used in almost all walks of life nowadays. Because of this, it would be impossible for us to maintain a comprehensive list of all careers in the field. However, we've put together a list of some of the most prominent careers below:

Average Salary: ~$105K/yr

Overview of Field:
Software Engineering is focused on the process of determining the needs of users, developing a list of problems to solve, and then planning a general solution for those problems. While all computer scientists do some software engineering, it is the job of Software Engineers to oversee the whole process and make sure every piece fits together. They often work more directly with clients than other programmers.

Keywords: Design    Class Diagrams    Environment    Analysis

Average Salary: ~$95K/yr

Overview of Field:
The most general type of programmer, Software Developers work at firms or for specific companies working on in-house software. While the exact work they do varies wildly by project, most of what these developers do is in an application programming setting. Whether they are designing tools for accountants or construction workers or doctors, software developers create software to help people.

Keywords: Programming    Development    Coding

Average Salary: ~$70K/yr

Overview of Field:
A popular field for computer scientists today, Game Developers create and implement systems that allow game designers to tell interesting stories and explore fun mechanics. From basic mechanics like character movement to robust combo attack trees, game devs are there to make these systems run fluidly so that the player can focus on the fun.

Keywords: Modeling    Game Engine    Prototyping

Average Salary: ~$98K/yr

Overview of Field:
The name of Full Stack Developers comes from their place in web development. These developers work on both the Front End (What your browser sees, HTML/CSS/JavaScript) and the Back End (Server/database). These developers use some of the widest arrays of languages, as they are working with different pieces of the system. There are also many frameworks built specifically to help with web development. These developers put all of the pieces of websites together and ensure the whole system works correctly.

Keywords: Front End    Back End    SQL    Web Dev

Average Salary: ~$100K/yr

Overview of Field:
Cyber security is all about protecting computer systems from unauthorized access, whether or not it is malicious or just accidental. Cyber Engineers are focused on locating potential vulernabilities in a system and adding fortifications to these areas so that hackers can't gain access. Because every industry these days uses computers, there is a massive need for people who can ensure that systems are secure and working as intended.

Keywords: Security    Cyber Crime    Survelliance
